# DNA Analyzer

A beginner bioinformatics project written in Python.

## Features

- Validates DNA sequences
- Ignores spaces in input
- Accepts lowercase and uppercase input
- Calculates sequence length
- Counts A, T, G and C
- Calculates GC content
- Detects invalid characters
- Handles empty input

## Example

Input

```text
ATGCGATTA
```

Output

```text
Length: 9

A: 3
T: 2
G: 2
C: 2

GC Content: 44.44%
```
